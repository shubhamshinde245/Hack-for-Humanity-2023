# Hack-for-Humanity-2023

### 2022 Winner - Roblox Sponsored Best Use of Gaming Technology
YouTube link : https://youtu.be/6l8_ydbWnWc

Project Description : 

Developed a game using Roblox Studio which can create awareness in people how small things which might be insignificant to them can create a huge impact if we multiply the same action with billions of people in the world. ( Lua programming language )
